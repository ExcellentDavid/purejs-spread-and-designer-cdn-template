# SpreadJS 项目模板

本模板包含了 `SpreadJS` 运行时 *Spread* 和 设计器 *Designer*，分别位于 `spread-template.html` 和 `designer-template.html` ，可根据项目情况酌情使用。

## 快速开始

安装依赖

```sh
npm install -g http-server
```

终端中进入项目路径，开启服务器

```sh
cd 你的项目路径
http-server
```
